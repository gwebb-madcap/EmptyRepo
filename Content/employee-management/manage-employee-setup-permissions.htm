Manage employee set up and permissions

Add and maintain employee information, log in credentials and access in Employee information.

The following guidelines help you avoid conflicts with Employee numbers:

  • If you are part of a store ownership, assign employees the same Employee number at all stores.

  • If using Phone Room, add the same employee in all stores where the employee is active to ensure the employee can print delivery invoices.

  • Do not reuse Employee numbers.

  • Do not edit a previously used Employee number to assign to a new employee to prevent listing the new employee on historical information completed by the previous employee, such as invoices.

Password guidelines

Only employees with Access allowed selected for Employee maintenance on the Security tab have access to the Employee information screen. The only security group with this setting is Store managers.

Add a new or edit employee and setup permissions

You can add new employee or make changes to existing employee information as needed. You cannot change the Employee number or reuse an existing employee information for another employee.

  1. From the menu, go to Employee mgmt > Employee information.

    The Main tab displays by default.

  1. To add a new employee, select New.

    To edit existing employee information, select the employee in the list or enter the employee's number in the Employee field.

  1. Enter the following required information:

    • Employee number Enter a number between 0 and 9,999 that has never been used for a current or past employee.

    • Job code Select the code matching the employee's job in the store, like Counter sales, Commercial sales associate, or Shipping supervisor.

    • Last name Enter up to 20 characters.

    • First name Enter up to 20 characters.

    • Pay type Select Hourly or Salary.

      If using time tracking in , employees with Pay type Salary are not required to clock in and out.

    • Profile number Required for employees with Pay type set to Hourly. The profiles are defined and maintained in Employee mgmt > Pay period profile maintenance.

    • Employee password Sets a temporary password. Employee will be asked to change it after logging in.

      You can use this field to reset the employee's password any time, but the employee is always asked to change it.

  2. If the employee needs access to the Point of Sale (Invoicing) screens, select the Invoicing allowed checkbox.

  3. Select in the Catalog preference field.

  4. If each of your employees has an individual A/R customer number to use when purchasing parts, enter the employee's A/R customer number in the A/R customer # field.

    If self-invoicing is not allowed, an employee cannot self-invoice when their A/R customer number is entered here.

    The Allow employee to self invoice checkbox on the Permissions tab of System mgmt > Invoicing > Invoicing controls determines if self-invoicing is allowed.

  5. Determine if the default timeout displayed in the Timeout minutes field is correct for the employee, and change it if necessary. Time out minutes determines how long the system is inactive or idle before locks the screen.

 Edit employee information

You make changes to an employee's information as needed. You cannot change the Employee number or reuse an existing employee information for another employee.

  1. Go to Employee mgmt > Employee information.

  2. On the Main tab, select the employee in the list or enter the employee's number in the Employee field.

  3. Enter or change information on any of the tabs and select Save .

 Remove employee access

If an employee resigns or is terminated, update the employee's information in to remove their access.

  1. Go to Employee mgmt > Employee information.

  2. On the Main tab, select the employee in the list or enter the employee's number in the Employee field.

  3. Enter the employee's last day in the Termination Date field and select Save.

On this date, the Invoicing allowed checkbox is cleared, and all the employee's security settings on the Security tab are set to No access.

You cannot delete employees from because the employee data is tied to transactions.

 Set employee security

Grant, deny, or give view access to menus and sub menus from the Security tab.

Keep in mind the following rules when assigning security:

  • You can grant permissions only as high as your own permissions. For example, if you have View only access to Finalize pay period, you can grant only View only access to other employees.

  • Security settings take effect when the employee logs out and then logs back in.

  • For an employee to access any sub menu options, you must give access to the main menu option. For example, to give an employee to access Employee sales fund detail, the employee must have access to Sales activities menu, and Review today's employee sales.

From the Security tab, select a menu or sub menu, and then choose one of the following options:

  • Access allowed Gives employee access to the selected menu.

  • No access Denies employee access to the selected menu.

  • View only Lets employee see the selected menu but does not allow action.

Set security using groups and copying settings

You can quickly set security for an employee by selecting a group and assigning those values or by copying one employee's settings to another employee. You can also manually define access to functions as necessary.

  1. Go to Employee mgmt > Employee information.

  2. On the Main tab, select the employee in the list or enter the employee's number in the Employee field.

  3. Select the Security tab.

  4. To quickly assign security settings, do one of the following:

    • Select a Group for the employee, and then select Yes on the warning message. All settings for the group are assigned to the selected employee, and any existing settings for the employee are replaced with the group settings.

      The security permissions for a group are hard coded and cannot be edited in . After the group permissions are applied to an employee, each setting can be changed.

    • Select Copy from employee, select Copy security, and then select Yes on the warning message.

  5. To modify individual settings, select the menu or sub-menu in the list, and then select the option for Access allowed, No access, or View only.

  6. To apply the same security level to a menu and its sub-menus, select Apply to sublevels, select the top menu option, and then select either Access allowed, No access, or View only.

  7. Click Save when all options are set appropriately.
