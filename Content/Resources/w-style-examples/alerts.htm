Note examples

To apply an alert, type your text, select the text and then select the div group from the Home menu, group icon, and choose the div style.

On all the alert styles, you cannot start with a bullet list. You need a stem sentence after the note heading and then you can start a bullet list

This is a best practice div style. Include a paragraph or bullet list of recommended tasks, etc. that we recommend for the task covered in this topic.

A best practice is a recommended order of operations, recommended strategy or recommended process for the task.

This is a prerequisite. Include a paragraph or bullet list of tasks the user should complete before starting the task covered in this topic, such as completing another task or printing a report.

A Prerequisite is a task or something the user needs before starting the task at hand. For example, the End of day process must be started before a user can balance the Cash received journal.
