Use the blue-header table style

  1. To add a table, select the Insert table icon from the Table ribbon.

  2. Do Not use the Insert table dropdown menu to choose number of columns and rows.

  3. Select 1 header row.

  4. Choose blue-header.css.

  5. Add your text. The table styles should automatically apply in each row.
