Modifying the video links

The video links are setup using the files in the Resources\Video folder. They are contained inside a table that is formatted using the RPM_video_container_tablestyle.css table stylesheet. This stylesheet is located in the Resources\TableStyles folder.

To add to a topic that does not currently contain videos:

  1. Add a 2-column table to the topic.

  2. Apply the RPM_video_container_tablestyle.css tale stylesheet to the table.

  3. Open the topic in the Text Editor.

  4. Paste the following code into the top of the topic after the <body> tag:

    <script id="bynder-widgets-js" data-account-url="assets.genpt.com" data-language="en" src="https://d8ejoa1fys2rk.cloudfront.net/bynder-embed/latest/bynder-embed.js"></script>

  5. Add the following code wherever you want to the left column:

    <div data-bynder-widget="video-item" data-media-id="D333248E-4C5A-4566-8538542E7F8A0963" data-width="" class="vidContainer" data-preload="none"></div>

  6. Change the data-media-id to match your video's ID.

  7. Type a description of the video in the right column.
