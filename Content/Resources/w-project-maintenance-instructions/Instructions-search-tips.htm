Modifying the search tips and popular search term list

The search tips and popular search terms are stored in snippets (search-tips and search-popular). These snippets are add to the bottom of each template page. When the topic loads, a function in scripts.js moves the two divs to the search bar. When the user clicks "Search tips" or "Popular searches," the matching div appears/disappears. Each div hides the other div (if it is currently visible) when it appears.

The labels "Search tips" and "Popular searches" are stored in the script.

To modify the search tips or popular search terms:

  1. Open the search-tips or search-popular snippet.

  2. Modify the snippet's content as needed.

To change the labels:

  1. Open scripts.js in a text editor.

  2. Modify the following:
    var searchLinks = '<p class="search-tips"><span id="searchtips">Search tips</span><span id="searchpopular">Popular searches on popular topics</span></p>';
