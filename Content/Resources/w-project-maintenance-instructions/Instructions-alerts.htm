Modifying the alerts

The Example-alerts topic contains an example of each alert.

Style rules

Each alert is created with two CSS rules:

div.note { background-image: url("../Images/alerts/note.svg"); background-position: top left; background-repeat: no-repeat; background-size: 40px 40px; padding-left: 60px; min-height: 50px; } div.note > p:first-child:before { content: "Note "; color: #3AB800; font-weight: bold; display: block; }

The main style rule sets the background image, padding, and min-height. The min-height is set so that smaller alerts do not "chop off" the image. The second style rule adds and formats the alert type ("Note" in the code snippet above) before the first paragraph in the alert. Note that this approach assumes that the alerts start with a paragraph. We are including a space after the alert type in case we change the formatting (for example, if "Note moves to the beginning of the note rather than above it).

To create an alert:

  1. Copy the two style rules for any existing alert.

  2. Rename the style rules.

  3. set the background image to the image you want to use.

  4. Change the alert's type.

  5. Change the alert's color to match the icon.

Icons

The icons are stored in the Images\alerts folder. We are using svg images so that we can set their width, height, and color using CSS.

To change an image:

  1. Name the new image to match the current image's filename.

  2. In Windows, replace the old image with the new image.
