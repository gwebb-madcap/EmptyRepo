Modifying the tabs

The tabs are created using a div with the "tab-content" class. The selected tab also uses a class named "current." The labels are created by a ul tag that uses the "tabs" class. Each list item uses a "tab-link" class and a data-tab attribute. The data-tab attribute (e.g., tab-1) should match its tab's number.

To add a tab:

  1. Right-click one of the existing "tab-content" divs and select Copy.

  2. Position the cursor where you want to add the new tab.

  3. Right-click and select Paste. The new tab will appear.

  4. Open the topic in the Text Editor.

  5. Change the new tab's ID. The tab IDs should be in order: tab-1, tab-2, etc.

  6. Ensure that only one tab (most likely, the first tab) uses the "current class.

  7. At the top of the tab code, locate the "tabs" bulleted list.

  8. Add a li for the new tab. The data-tab attribute should match the new tab's number.

To remove a tab:

  1. Right-click the tab in the block bar and select Delete.

  2. Ensure that the tabs' IDs are unique and increment: tab-1, tab-2 etc.

  3. Click inside the tab's label.

  4. Right-click the label's li tag in the span bar and select Delete.
