Modifying the home page cards

The home page topic uses the home.css stylesheet.

Each area of the home page is contained in a div: homeWel, homeOp, and homeTrain. All of them are setup to use a 3-column grid layout. The divs are defined in the screen medium so that Flare does not display error messages. Flare's editor does not support the grid display property, so the content will not appear in columns in the editor.

Each div has a negative top margin so that the top border (if it exists) will overlap the section's heading (an h1). The h1 should use a gradient background (like the current homeOp h1) to match the background colors.

homeWel

The homeWel div currently contains three divs that are formatted to resemble buttons. They are formatted using the "homeWel" selectors.

To change a welcome link or its text:

  1. Click inside one of the three divs.

  2. Type the new text (if needed).

  3. Right-click the a block in the span bar and select Edit Hyperlink.

  4. Change the link as needed.

homeOp

The homeOp div currently contains 9 "cards." Each card is a 2-column grid. The left column's width is set to 90px and contains a span with a background image (the icons). Each span uses the "icon" class and a unique class name. The icon class sets the icons' width, height, and background color using a filter. A hover pseudo class swaps the background color (also using a filter). This approach allows us to control the svgs colors and avoid us having to have separate default and hover images. However, the icons will not appear in Flare's editor.The right column is set to use the remaining width of the card. It contains two spans: title and description.

To change an icon:

  1. Find the icon's class in the stylesheet.

  2. Check the name of the icon's background image.

  3. Edit or replace the background image svg file.

To add a card:

  1. Right-click a card's div tag in the block bar. The div is highlighted.

  2. Select Copy.

  3. Move to where you want to add the new card.

  4. Paste the new card's div.

  5. Change the title as needed.

  6. Change the description as needed.

  7. Open the topic in the Text editor.

  8. Change the icon's class to a new unique class name.

  9. Save the home page topic.

  10. Create an icon for the new card.

  11. Open the stylesheet in the text editor.

  12. Add a style class for the new card's icon.

homeTrain

The homeTrain div contains three divs. The first two divs contain a title and bulleted list. The title formatting is the same as the other section's titles. The bulleted list is formatted using the div.homeTrain li and div.homeTrain li::marker selectors. The third div contains a button. The button is currently formatted to match the buttons in the Welcome section. However, it uses unique selectors: div.homeTrain > div.c2a p, div.homeTrain > div.c2a p:hover, and div.homeTrain > div.c2a a.
