When your stock gets low, to ordering you go! Stock replenishment (also known as restocking or inventory replenishment) ensures that you are reordering the right stock items to meet your customers' demand.

Create your Order selection groups before you order.

Create order

When you are ready to order stock, create your orders.

  1. From the menu, go to Inventory > Purchase orders > Create order.

  2. Select your Order type and Order selection group.

    Add sample screen

  3. If necessary, modify any fields, and select Create order.

    If you make any changes, prompts you to save your changes to the order selection group. Click Yes to save these changes for future orders that use this group, or click No to use your changes for this order only.

  4. displays the order, including the part, line number, description, class, order quantity, value, and how many you have on hand. The CD column may contain the following symbols:

    • @ Part is on order

    • ! Economic order quantity

    • & Extra discount used

    • * Min/max amount used

  5. Select a part in the order to display additional information on the bottom half of the screen.

  6. Use the information to determine if you should edit the order quantity (Order qty).

  7. Use the Special edit button to change the Min or Max quantities if the part is selling faster or slower than in the past.

  8. Click New to add additional parts to the order, or select a row and click Delete to remove a part from the order.

  9. If parts are listed with 0 as the order quantity, review the reason code to determine why.

  10. If a part has been superseded, select it and click View old parts to see additional information.

  11. Click Print order to view a working copy of the purchase order in the Report Viewer before finalizing it.

  12. When you are satisfied the order is correct, click Print order, select Finalize order, and then click OK.

The order is transmitted when you print and select Finalize order.
Reaccess an order

You can reaccess an order if you have to exit order creation before you finalize.

  1. From the menu, go to Inventory > Purchase orders > Edit order.

  2. Select the order in the list and edit as necessary.
