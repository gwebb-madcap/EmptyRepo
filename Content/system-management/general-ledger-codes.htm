General ledger codes

General ledger codes categorize income and expenses in logical groups to record money flowing in and out of your store. General ledger (G/L) codes are a unqiue code or number and description of each type of expense, like 140 Inventory, 810 Repairs and maintenance, or 820 Taxes other.

Use G/L codes to track recurring expenses, reconcile vendor statements against actual invoices for buyout purchase orders, and view amounts charged against each expense type in the Cash received journal.

General ledger (G/L) codes represent the balance sheet and income statement accounts in your general ledger's chart of accounts. Transactions and other financial amounts are summarized and rolled up into these accounts for reporting purposes. Examples of G/L account codes are Cash, Cash-in-Bank, Accounts payable, and Accounts receivable.

G/L codes are set up by APAR.

Required if using to Balance Cash received journal and Buyout purchase order process. General ledger codes are company-specific and all stores in a store ownership group use the same codes.

Add general ledger codes

General ledger codes are required to use to Balance Cash received journal and complete the Buyout purchase order process.

Are permissions needed to set up? 

Create general ledger (G/L) codes for each type of expense, like 140 Inventory, 810 Repairs and maintenance, or 820 Taxes other.

  1. From the menu, go to System management > G/L codes. The G/L codes screen is where you add new G/L codes.

  2. To add a new G/L code, select New .

Can you help so that I can add a code

  1. Enter information:

  • G/L code Required. Unique number for the code. Can you enter duplicate? 

  • Description Required. Clearly describe the purpose of the code. For example, stationary and supplies.

  • Valid for cash journal Select to use this code when balancing and finalizing the Cash received journal.

  • Valid for buyout Select to use this code to complete the Buyout purchase order process.

  1. Select Save .

  2. Repeat to add all G/L codes used by your store.

Edit G/L codes

  1. From the menu, go to System management > G/L codes.

  1. Select the G/L code, and then select Edit .

    How do you edit? That icon adds a note

  1. Make your changes to the following:

  • G/L code Unique number for the code.

  • Description Clearly describe the purpose of the code. For example, stationary and supplies.

  • Valid for cash journal Select to use this code when balancing and finalizing the Cash received journal.

  • Valid for buyout Select to use this code to complete the Buyout purchase order process.

  1. Select Save .

To delete a code, select the code. Select Delete , and then confirm.
