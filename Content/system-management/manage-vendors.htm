Required to maintain up-to-date vendor list.

When customers need parts outside the supply chain, maintain vendor information in to create the buyout (also called pickup) order. Because you sell this merchandise immediately, or it is supplies for your store, it is excluded from your on-hand inventory. To order from these vendors, you must add them to your vendor list so you can create a buyout purchase order. From legacy help, Is this true? 

Add vendors as soon as you establish a buying relationship.

Future content on Order selection groups point to the vendors from whom you order merchandise. For each group, you can enter vendor contact information on the Vendor Information tab.

Is Supplier code a pre requisite? This from legacy help: "Supplier codesYou also set up supplier codes for the vendors who sell you merchandise that you normally keep in stock. NAPA reserves code 0 (host NAPA Distribution Center) and 1 (NAPA Factory Direct), but you can assign codes 2-255 to your other suppliers."

Add a vendor

  1. From the menu, go to System management > Vendors.

  2. Select New .

  3. Enter the following required details.

  • Vendor number Numeric code (15 numbers) to identify vendor. Are there restrictions or requirements? 

  • Vendor name Company or business name.

  • Alpha key Alpha or numeric name (up to 6 characters) for vendor. Where does this show up?

  1. Select Save .

 Modify vendor

  1. From the menu, go to System management > Vendors.

  2. Enter the Vendor number or Search to find the vendor.

  3. Edit information.

  4. Select Save .
