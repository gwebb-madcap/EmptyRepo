Maintain vendor screen

Required to maintain up-to-date vendor list.

Use the Maintain vendor screen to add and edit vendors and their information.

Vendor number Numeric code (15 numbers) to identify vendor. Are there restrictions or requirements? 

Vendor name Company or business name

Contact First and last name of the primary contact.

  • Alpha key Alpha or numeric name (up to 6 characters) for vendor. Where does this show up? 

Federal id number Vendor's federal taxpayer id number.

Address 1, 2, 3, 4 Up to 4 lines for the vendor's street address.

City, Country, State, Zip, Phone number, Extension number, Fax number Vendor mailing and contact information.

Speed dial Code number used to speed-dial the vendor's telephone number.

Web address If available, enter URL for the vendor's Internet address.

E-mail address If available, enter the vendor's Internet e-mail address. For reference only, emails cannot be sent to vendor?
