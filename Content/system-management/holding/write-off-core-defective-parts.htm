Company owned store process

Company stores work with the APAR center to reconcile outstanding core balances as follows: 

  • Write off invoice numbers update during the End of day or when you process commication transmissions. Instead of legacy help: TAMS II receives the write-off invoice numbers from the APAR Center during communications.

  • Invoice numbers received during the communication are matched to the invoice number from the core bank.

  • Line items linked to the invoice number update to Write-Off status.

Write off cores

  1. From the menu, go to Communications > Process received transmission.

  2. Select Write off core tracking.

  3. Select Process.

 Independent store process

Independent stores not working the APAR center reconcile outstanding core balances as follows:

TB done - walk through process

  1. Open cores for the current customer can be viewed while creating a journal entry.

  2. Click View Open Cores. The following information appears:

  • Invoice Number, Core Price, Date of Invoice, Line Abbreviation, Part Number, Part Description, and the total number of outstanding cores, and Extended Core Price.

  • You create a credit journal entry and then apply the credit through the Applying Payments process for an invoice transaction with open cores.

  • The system prompts if the payment/credit application should close all cores for the transaction. Click Yes to confirm.

  1. TAMS II updates the status of all line items linked to the purchase invoice number to "Write-Off."

Unapply Write-off

To change the status of a transaction from Write-Off to Open, select the transaction and click Unapply.
