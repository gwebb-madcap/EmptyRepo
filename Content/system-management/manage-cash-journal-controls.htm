Manage Cash journal controls

To use the Cash received journal for reconciling and balancing daily transactions, you must enable the Cash received journal and set up appropriate general codes from your accounting system.

Cash journal controls are specific general ledger codes from your accounting system that account for specific transactions. For example, For example you may have a charge customer who paid using a check. Or, you may have given customers a discount for making early accounts payments. Or, you may have merchant fees associated with certain payment methods.

Are there any best practices? 

Before setting up cash journal controls: 

  • Add General ledger codes from your accounting system so they are available as Cash journal controls.

  • Assign general ledger codes to categories in What are these categories and how do they assign them?  

Assign general ledger codes

  1. From the menu, go to System management > Invoicing, and then select Cash journal controls.

  1. To enable Cash received journal, select Cash journal in use.

Lock prevents unauthorized users from clearing or changing Cash journal controls. Once locked, you must contact support to unlock the field to make changes to it.

  1. Select the following required general ledger codes:

    • A/R G/L code Code from your accounts receivable system? From legacy help: "Enter G/L code for A/R" What is this one for? 

    • Cash G/L code Code used for cash, check, and credit card amounts you collected for the business day.

    • Payment discount G/L code Code used for the discount you offer customers for paying statement balance early.

    • Merchant fees G/L code Code for the merchant fees your store pays for credit card processing.

    • AR trade G/L code Product Please help with definition

    • AR cash sales G/L code Code from your accounts receivable system for customers set up with a charge account but are paying with cash.

  2. To account for funds being deposited to the bank, select To office deposit. This adds it as an additional Deposit type when balancing the Cash received journal.
