Create and maintain customer notes

Customer notes are a way of adding specific information to a customer's invoice. For example, if a customer has specific delivery instructions about where to leave parts, you can add it as a note. Another example is a customer may want a specific employee's name to display in the attention field on invoices. Add note like Add attention to shop employee as a reminder to users invoicing customers.

  1. From the menu, go to Accounts receivable > Customer notes.

  2. Select New . The Note number is automatically assigned.

  1. In Note, enter a name for the note.

  2. In Note details, select a Note handling option: 

  • Screen and invoice Displays on the Invoices screen and prints on receipts.

  • Screen and invoice (must acknowledge) Displays on the Invoices screen, prints on receipts, and requires a user to acknowledge the note.

  • Screen only Displays on the Invoices screen only.

  • Screen only (must acknowledge) Displays on the Invoices screen and requires a user to acknowledge
