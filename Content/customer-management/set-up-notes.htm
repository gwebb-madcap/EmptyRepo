Set up customer notes

Customer notes are a way of adding specific information to a customer's invoice. For example, if a customer has specific delivery instructions about who or where to leave parts, you can add it as a note. Once notes are added, assign them to each customer as needed.

  1. Add notes

  2. From the menu, go to Accounts receivable > Customer notes.

  3. Select New .

  4. Select where to the note in Note handling:

  • Screen only Displays on the Invoices screen only.

Screen only (must acknowledge) Displays on the Invoices screen and requires a user to acknowledge

Screen and invoice Displays on the Invoices screen and prints on receipts.

Screen and invoice (must acknowledge) Displays on the Invoices screen, prints on receipts, and requires acknolwegement.
