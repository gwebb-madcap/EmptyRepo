Cancel BOPIS orders

Integration messages let you know when a customers cancel online orders.

  1. From the Invoice screen, go to Message. Select Integration messages, and then select OK.

  2. Review the message, and then select Acknowledge.

  1. Select the printer, and then select OK.

    • A message lets you know a pull list is printed so you can return the part to its location.
