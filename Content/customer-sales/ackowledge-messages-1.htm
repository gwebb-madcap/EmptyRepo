Review and acknowledge messages

  1. From the Invoice screen, select Message. The Review messages screen shows the message types.

  1. Select the message type to acknowledge, and then select OK.

  1. After reading the message, select Acknowledge. Use Next message to continue when there are multiple messages. Your Employee id is associated with messages you acknowledge.

  2. Select Close when you are done.
