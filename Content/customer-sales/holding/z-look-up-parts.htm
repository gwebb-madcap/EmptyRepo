Look up parts

  • Overview

When you need help finding parts, use the catalog directly from a customer's invoice. is the complete online catalog with a comprehensive but easy-to-use Search. Search works much like a Google search. Simply enter a keyword or phrase, and then use the filter options to refine the results. Once you find the part, add it to the cart. Return to the customer's invoice and checkout as normal.

Open

  1. To launch , be sure you have a web browser open a web browser.

  2. From the Invoice screen, go to Catalog.

  1. Use one of the Search types described below to find parts.

  2. Once you find the part, enter the quantity, and then select Add to cart.

  3. Go to the customer's invoice and checkout as normal.

Search types

Each Search type is described below.

Quick search

Use to find simple parts, like air filters. Enter as few or as many words in the Search bar, like 2014 Jeep Patriot air filter.

Select the result that best matches the part. You can continue to refine and sort results until the specific part is located.

  • Refine Results by Narrows results further by the option selected.

  • Sort by Change display order.

 Search by vehicle application

Use Vehicle application when you know the vehicle identification number, make, or model.

Refine the search using a Search by option.

  • Keyword Use to find a single part, like an air filter or spark plugs. Select the Vehicle conditions and Transmission type.

  • Job type Use to find all the parts needed for a specific job, like brake job-complete. Select the Year, Make, and Model.

  • Category Use to search by Category, like Ignition parts & engine filters and Subcategory, like Ignition parts, distributors & distributor parts. Select the Year, Make, and Model. Then select a Category and Subcategory.

 Search by product specification

Use Product specification to refine your search results when the customer does not know specifics about a part, like radiator hose size.

  • Enter a keyword to Search based on product specifications, like hose.

  • Or, select a Category, Subcategory, and Part type.

Recent searches from your store

Use Recent searches to find parts based on a previous search. You can use one of the last 25 searches used for this customer.

  • Me Searches you used to find parts for this customer.

  • All Searches other store users used to find parts for this.

 Search by NAPA equivalents for competitor parts

Use to search by competitive parts:

  • Interchange Searches equivalents for competitor parts.

  • Reverse interchange Searches competitor equivalents for parts.
