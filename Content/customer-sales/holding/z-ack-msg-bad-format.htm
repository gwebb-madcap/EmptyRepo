Messages are notifications for incoming orders, returns, or communications from other systems. Messages remain pending until they are acknowledged. Once acknowledged, a message is ready for an action, like creating an invoice, sourcing a part, returning a part, viewing the sales history, or processing a communication. Your Employee number is assigned to messages that you acknowledge.

Review and acknowledge messages

  1. From the Invoice screen, select Message. The Review messages screen shows the message types.

  2. Select the message type to acknowledge, and then select OK.

  3. If a customer has several messages, you are asked to consolidate them. Select Yes if you would like to combine the messages into one.
