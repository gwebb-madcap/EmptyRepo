Check out and tender payment

After staging parts to fulfill a customer's invoice, you are ready to checkout and tender payment. On-hand inventory quantities update when you finalize the invoice.

  1. From the customer's invoice, select Checkout.

    To open a saved invoice, select Recall saved invoice.

    • Select the invoice, and then select Invoice.

    • Select Checkout.

      I do not have quantities on-hand? When your store does not have sufficient quantity to fulfill the invoice, a message asks you to source the parts now. Select Yes to Source parts from NAPA Xpress.

  1. For delivery orders, select Delivery and complete the following.

    • Charge Add a Delivery fee if needed.

    • Priority Assign a priority delivery code. The following delivery Codes and Names are standard. Change the delivery times as needed.

      • Code Name Example time in mins
      5 Default 120
      6 Platinum 30
      7 Gold 45
      8 Silver 60
      9 Bronze 75

  • Date If not using a Priority delivery code, enter a Date (mm/dd/yyyy), Anticipated time, and Method.

  1. Select a Payment method and enter the following details.

    • Cash Enter Amount received from customer. Change is automatically calculated.

    • Charge If a charge account is set up in the customer's account. the total is automatically billed against the account.

    • Check Enter Check amount, Check number, and Customer name.

    • Payment card If a credit card is set up in customer account, Enter Payment amount, Card on file, and Card type. Have these been removed: last 4 digits, and Authorization number are required.

    • gift card Enter Payment amount, last 4 digits of card, and Authorization number.

    • What if customer is using multiple payment types

      Multiple tender types must be enabled on the customer's account to tender more than one method. Select the first payment method and enter the amount. Select Submit, and then repeat for each type.

  2. Select Submit to tender payment.

  3. Enter a name or note about the invoice in Attention, like Bill Smiles will sign. What is an example of how to use this

  4. If the customer is using a purchase order number, select the PO number.

  5. Select the Printer to print the invoice.

    • Select Print to complete checkout.

    • Select OK.

    • A PDF invoice is automatically sent to the customer's email. Cannot see this to verify how email functionality works, was told to use server 108 for 3.07 functionality

      If you have multiple printers, like Commercial accountsor Cash customers, be sure to select the correct printer to avoid dispatching duplicate parts.

Source parts from NAPA Xpress

When customers need parts not in your on-hand inventory, order them through .

  1. From the customer's invoice, select Checkout

  2. Select Yes to order from .

  1. To change the recommended Fulfillment method, select Edit fulfillment, and then select one of the following:

    • Default Default fulfillment method set for your store.

    • Priority Best price and time.

    • Fastest Quickest fulfillment time.

    • Grouped Use to reduce the number of shipments when ordering multiple parts.

  1. For parts not sourced locally, change the shipping method, cost, and estimated delivery and change as needed.

  2. Add the customer name and contact details.

  3. Review the subtotal, and then select Place order.

  1. To save the invoice until the parts arrive, select Special functions > Save invoice.

    To track the order status Go to Invoicing  Messages > NXP order history.

    To recall the saved invoice Select Recall saved invoice F7. Select the invoice from the list. Select Invoice.
