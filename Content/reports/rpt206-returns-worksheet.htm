RPT206 Returns worksheet

Required to use report when shipping class returns.

The lists parts to pull for return to the distribution center. It also shows the current and suggested min/max values and sales history to determine the quantity of the item to return.

IMAGE OF REPORT

The report is divided into sub-sections by product line and stocking class. The class line detailed information in each of these sub-sections includes:

Keep quantity

Return quantity

Part number and description

Reason code

Group code

Class

Per carton quantity

Standard package quantity

Factory Order flag (this is a Y/N flag indicating whether this part is order from the factory)

Min/max status

Current min/max amounts

Initial stocking date

Suggested min/max amounts

Current sales including the current and previous 12 months

Previous Lost Sales Information

Superceded parts, description, and line abbreviation print below the part details.
