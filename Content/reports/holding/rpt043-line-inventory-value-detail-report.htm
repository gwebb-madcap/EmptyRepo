RPT043 Line inventory value detail report

Do we need to cover this report in v1? 

Optional report

provides detailed information for each line in the inventory along with month-to-date and year-to-date price changes and inventory adjustments.

Oversold parts are not included in the total inventory value. They are only listed in the value of oversold parts.

Report Layout

This report provides the following detailed information for each line in the inventory:

Need to define each

  • Line Abbreviation Group Code

  • Description Class

  • Stocking Depth Return Standard Package (Y/N)

  • Minimum Gross Profit %

  • Date of Last Physical Inventory

  • Date of Last Classification

  • Date of Last Price Change

  • Date of Next Price Change

  • MTD Price Change amount

  • YTD Price Change amount

  • MTD Price Adjustments

  • YTD Price Adjustments

  • Total Number of Part Numbers

  • Total Replacement Value

  • Value of New Cores

  • Value of Returned Cores

  • Value of Returned Defectives

  • Total Average Cost Value

Within each product line, parts that have been oversold, have no cost, or whose value exceeds the report high value are listed. A memo section prints the month-to-date and year-to-date price changes and inventory adjustments. The last page of this report provides a recap of the following information for NAPA parts, non-NAPA parts, and all parts in the inventory:

  • Total replacement and average value of stocked inventory

  • Total replacement and average value of items with zero minimum stocking level

  • Total replacement and average value of new parts

  • Total value of new cores

  • Total value of returned cores

  • Total replacement and average value of defective parts

  • Total number of parts

  • Total number of stocked parts

  • Total number of items with zero minimum stocking level

  • Total month-to-date and year-to-date price changes and inventory adjustments

How to use This Report

This report can be used to investigate an inventory value discrepancy. When reviewing multiple reports at the same time, you may be able to identify specific areas of inventory value rise or fall.

You can also use this report to justify your inventory investments by monitoring the inventory of a specific line code. You note what section of the line code the change is in, and then run RPT111 to complete your research.

Selection Criteria You can sort this report by High Value. To do so, enter the high value dollar amount. High value is compared to part cost multiplied by quantity on hand. If the dollar amount is greater than the high value, that part is listed on the report as exceeding the high value.
