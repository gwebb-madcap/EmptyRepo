Required to use report to verify daily changes to inventory and product lines.

What's on the report?

The report shows the following details for

Inventory and product line details

Shows the following information: 

  • Cash sales for the product line.

  • Employee (number) who made the adjustment.

  • Field adjusted.

  • Original value before changes were made.

On-hand quantity adjustment summary

Shows the following information:

  • Cash sales for the product line.

  • Employee (number) making the adjustment.

  • Process in from which the change was made.

  • On-hand before is the value before adjustments were made.

  • On-hand after is the current value after adjustment.

  • On-hand adjusted is the value before adjustment - value after adjustment).

  • Cost including core cost), sum (sum of adjustment amount x the cost), and total for the product line.

On-hand quantity adjustment recap

  • Shows the following infomration:

  • Process in from which changes was made.

  • Amount including sum of extended adjustment amount for each on-hand adjustment, total adjusted (sum of all cash received).

Print the report

The report prints as part of the End of day process. To reprint the report: Is this how they schedule report? 

  • Go to Reports > Reports manager, and then select EOD from the Report group dropdown menu.

  • Select the report, and then select Selection criteria.

  • Enter a date to generate the report on the Report timeframe screen, like EOD or EOM, and then select OK.

  • Generate the report for the Current accounting day in the End of day - selection criteria screen.

  • To print the report, select View report, and then select the printer.

  • Select OK.
