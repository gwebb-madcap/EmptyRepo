When customers need parts or merchandise outside the supply chain, the buyout purchase order process manages the sale from ordering through the vendor to reconciling the purchase order. The buyout purchase order process (also called pick up order) is a multi-step process requiring you to: 

  • Create a buyout purchase order to generate a purchase order number for the vendor.

  • Invoice the customer to tender payment.

  • Complete the purchase order once payment is received.

Create a buyout purchase order

The first step is to generate a purchase order number to provide the vendor to purchase the part.

  1. From the Invoice screen, enter the customer account number (customer requesting the merchandise).

  2. Create the buyout purchase (pick-up) order:

  • Go to Special functions > Pick-up.

  • A message lets you know when customers do not have buyout purchase orders. Select OK to continue.

  • In the Customer buyout screen, select Add po.

  • The Buyout purchase orders screen shows open buyout purchase orders.

  1. To create a new buyout purchase order, select New .

  1. Enter the following details in Create buyout po screen:

  • Vendor number Numeric code (15 numbers) to identify vendor.

  • Payment type Select Cash, On account, Purchase card.

  • Purchase card number Manager's purchase card (p card) number.

  • Select OK to generate the buyout purchase order number. Give this number to the vendor.

  • Select Add detail.

    The Buyout purchase orders screen opens where you add part information.

  1. To start adding a new part, select New. Enter the following information:

  • Type Select Special order.

  • Part number Enter the buyout part number.

  • Line Select a non- product line.

  • Description Describe the part using up to 21 alpha-numeric and special characters.

  • Order quantity Enter the quantity you are purchasing from the vendor.

  • Unit cost

  • Core cost

  • Customer number Enter or search for the customer's account number.

  • Invoice number and Invoice date are blank until you tender the invoice

  • Select Save .

 Create a customer invoice

The second step is to create a customer invoice so you can bill the customer and tender payment for the part.

  1. From the Invoice screen, enter the customer number.

  2. Select Special functions > Pick-up.

    The Customer buyout screen shows the open buyout purchase order.

  • Select the purchase order, and then select Next.

  1. The Edit buyout po screen shows the part information.

  • To change the Quantity or cost, select the part and enter the correct value.

  • Select Next to continue.

  • To create the invoice, select Invoice parts.

  1. If the Pickup gross profit percent is set up, the sale price is based on the cost entered on the buyout purchase order. Enter the price for the part, and then Check out as normal.

Complete the buyout purchase order

The final step is to complete the transaction and match the part purchased with the part ordered and the part sold.

  1. From the menu, go to Inventory> Purchase orders > Buyout PO.

  1. Select the open buyout purchase order, and then select the Other options tab.

  1. In Vendor invoices, select Add.

  2. Complete the following vendor invoice information: 

  • Invoice number Invoice number provided by the vendor. Is this correct? 

  • Invoice date Date you are completing the buyout purchase order.

  • Amount Price you are charging customer for the part.

  • Payment type Select your method of payment: Cash, On account, or Purchase card.

    If your purchase card is on file, select it from the dropdown menu.

  • Select Save.

  1. In G/L information, select Add. Complete the following:

    Required to enter G/L information:

  • G/L code Select 140 Inventory as the general ledger code to associate with the part.

  • Amount Enter the amount store pays for part before tax.

  • To add taxes, select Add.

  • G/L code Select the tax code (typically Taxes).

  • Amount Enter the tax amount.

What if The part does not have tax?

  • Add a general inventory code using code 140 Inventory, and round the amount down by 1 penny (for $20.00, enter $19.99).

  • Add a tax code using 820 Taxes, and enter .01. This adds a tax line to the invoice.

Company stores

This option is enabled by default; the below process does not apply.

When you do not want to add a buyout part to your on-hand inventory, create a buyout purchase order from the customer's invoice.

Clear Buyout purchase order in use. Where is it? System management > Inventory controls > Buyout po > Buyout po controls.

  1. From the Invoice screen, enter the customer number.

  2. Select Special functions > Pick-up.

  3. Enter the following information:

  • Part number Part ordered from supplier. Part does not have to exist in .

  • Line Enter 3-character abbreviation for non- product line.

  • Description Enter a description for the part.

  • Quantity Total quantity ordered.

  1. The Pickup screen opens where you enter the following part details: 

  • Pick-up item cost Store cost for the part.

  • Do you wish to enter a lost sale for the item? Select Yes if you purchased the part because you could not get the equivalent on time to make the sale. Otherwise, select No.

  • Select OK to return to the customer's invoice.

  1. If Pickup gross profit percent is not set up, enter the sale price of the item in PD on the customer's invoice. If Pickup gross profit percent is set up, the sale prices is automatically determined.

  1. Tender and checkout as normal.
