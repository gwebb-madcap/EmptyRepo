distribution center runs 4 rounds of classifications each year. Is this accurate? 

Classification returns (class returns) refer to parts that are excessive or slow-moving. Your store receives quarterly class return file from the distribution center. Once you receive a class return, remove the parts from inventory, and ship them to the distribution center.

Instruct multiple users to enter different line codes on their RF scanner to work the class return at the same time.

Pull classification returns

  1. Log into the RF scanner, scroll to Class pull, and then press Enter.

  1. Select the number of the Class returns session you are pulling from the Return worksheet.

  1. Select the parts to review:

  • All New class returns.

  • Variance only If you have not pulled parts, all parts are in variance. Variance only occurs when you have not started, did not pull suggest return amount, or pulled more than the suggested return amount.

  • Partial pull Resume an interrupted session.

  1. Scan each part in the class return to update the returned quantities.

    For manual entry, use the scanner input field.

I cannot find a part? Press the Down arrow to move to the next part.

  1. After scanning all parts on the class return, press Esc to exit the scanner session.

Product owners, please provide steps to back out and return to a session that cannot be completed at once.

  1. Let your coordinator know when you finish pulling parts. Coordinates review and approve the class order before the class file is finalized.

Once class returns are finalized, you cannot make changes.
