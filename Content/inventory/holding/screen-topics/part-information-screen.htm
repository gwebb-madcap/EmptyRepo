Min/max status

The status of the minimum and maximum settings: Help with better description.

N Normal Part follows the normal Min/max processing what does processing mean? and uses sales history to set Min/max and stocking levels.

L Locked Keeps the Min/max values set at the current setting and does not evaluate or change Min/max values. Product supersede or obsolete overrides when Locked. When the status is Locked you cannot update or generate new Min/max values for the part.

K Keep (Keep Flag) Keeps a positive Min/max value, keeps the part as a Stocked or In class. The Min/max values can change, but will not go to zero unless the part is obsolete or superseded. The Keep option affects the following items:

  • Classification When performing classification, parts with Keep status are always evaluated and considered as In Class, unless they are Obsolete.

  • Min/max generation Calculates suggested Min/max values when the status is Keep. The Min/max values for the part can be updated.

If the status is set to Keep and the Min/max calculation generates a suggested Min/max of 0, the suggested Min/max value is Per car.

The Min/max is not 0 when a part is set to Keep.

A 0 Min/max means the part is no longer stocked. Set Min/max to Keep, to keep the part in stock.
