End of day screen

Use the End of day screen to prepare and start the process to end the current business day.

From the menu, go to Sales activities > Start EOD.

Fields on the screen

The following table lists the areas and fields on the screen with a description of each.

Area

Field

Description

Checklist

 

Select and perform various tasks before starting the End of Day process. Complete tasks without a check mark in the order shown.

 

List

Name of the task.

 

Status

Checkmark indicates task is complete. Blank means task needs an action.

 

Action

Updates status of task to complete or skipped.

 

Bin labels exist

This choice is available when bin labels are available for printing.

Generates price change bin labels automatically as part of the End of day process, regardless of Print bin Labels at end of day setting on the Inventory management controls screen.

 

Review report selection

Lets you review selection criteria for printing End of day reports.

 

Major account pricing corrections

Lets you review and correct major account pricing that have not been processed.

 

Review employee time

Lets you review employee time tracking information. A reminder lets you know if today is the last day of the current pay period, so you can finalize (close) the period if needed.

 

JOEI orders exist

Lets you transmit pre-loaded JOEI orders.

 

order has not been created

If , non-, special order, or direct ship stock orders exist, you can create these orders.

 

Orders exist that have not been finalized

Lets you print and finalize open purchase orders.

 

Branch store invoices exist

Lets you post (update) branch store invoices. If your store is a branch store, process today's invoices, and send the branch transactions to the main store.

Store transmission files exist

 

If transaction transmission files exist, you can update and transmit the files.

EOD information

 

Shows messages about the current business day and month.

 

Your month-end Date is set to

Last date of the accounting month.

 

End of day processing will run for accounting day __the calendar day is

Number of the accounting day, and the calendar day of the month.

 

Your daily backup will start at

Time when the tape drive will be begin to back up your files.

 

Insert tape number __ for [day of the week]

Day of the week and tape number to insert in the tape drive.

 

 

 

Start EOD

 

Start the End of day process.

Cancel

 

Return to previous screen without starting the End of day process.
