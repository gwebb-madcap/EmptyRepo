End of day monitor screen

Use the End of day monitor screen to track the End of day process. The End of day monitor opens when End of day starts. You can also open the monitor manually:

From the menu, go to Sales activities > EOD monitor.

Once started, you cannot stop or interrupt the process.

Fields on the screen

The following table lists the areas and fields on the screen with a description of each.

Area

Field

Description

General information

 

Provides status information about the End of day process. identifies the date and accounting day of the process is currently running, user who started End of day, and the computer where it is running.

 

End of day running at

IP address of the computer running End of day.

 

End of day started by

Name of user who started End of day.

 

End of day accounting day

Number of the accounting day being closed.

 

End of day started

Calendar day and time when End of day started.

 

Last completed program

Last completed process.

 

Program end time

Time the last process finished.

 

Currently running program

Process currently in progress.

 

Program start time

Time current process started.

Log

 

Status messages about End of day process, like End of day collection number __ has been processed with no errors.

 

Start backup now

Starts the backup after End of day processes are complete instead of at the pre-scheduled time.

 

Restart

If End of day does not complete successfully, select Restart start it again. You can restart when End of day is not running on the system.

Start backup now is available when two of the following conditions are true:

The End of day monitor screen is open on the computer where you started End of day.

From legacy help, is this still true:

 The communications processing End of day is running on the system.

The T300 system running Solaris10 has been certified to support the use of USB flash drive media when processing a TAMS II backup.

When more than one media is available to perform a back up, the system uses the supported media devices in the following order:

USB flash drive media (Reformatted each time it is used, so only one backup exists at a time.)

Tape drive

DVD drive
