Log in and out

Use your employee number and password to log into .

  • Set up security questions and answers. This makes it easier to retrieve or reset your password.

  • Set a strong password using guidelines for creating passwords.

  • If you work at more than one store, ensure you are set up with the same Employee number and password at all stores. Seek your manager's assistance if needed.

First-time log in

Make sure you have a temporary password and know your Employee number. If not contact your manager.

  1. Enter your Employee number and the temporary Password, and then select OK. If you do not have either, contact your manager.

  1. Click OK on the warning to change your password.

  2. Set up your security questions.

  • Select a security question and answer in both fields.

  • Select Accept, and then select OK on the confirmation message.

  1. On the Change password screen, enter and confirm your new password using guidelines for creating passwords.

    Select Accept.

    The store number and your name show next to the logo on the menu screen.

If you enter your password incorrectly five times, your account is locked for 10 minutes. A manager can reset the password on the Employee information screen; you will be asked to change it when it is reset.

 Set up security questions

  1. From menu, select Log out > Logout/in.

  2. Select Change password, and then enter your current password in the Enter old password field on the Change password screen.

  3. Select Set security answers.

  4. Select a security question and answer in both fields.

  5. Select Accept, and then select OK on the confirmation message.

 Reset your password

You can reset your password at any time, and a password reset is required when the Expires in days field on the Employee information screen has been met.

displays a warning message 15 days before your password expires.

  1. To change your password from the menu, select Log out > Logout/in.

  2. On the Logout screen, select Change password.

  1. Enter your Old password.

  2. Enter and confirm your New password.

  3. Select Accept.

Forgot your password?

Use the Forgot your password feature to reset your password when you cannot remember it.

  1. On the Login screen, enter your Employee number.

  2. Select Forgot password.

  3. Answer the security question and select Accept.

  4. On the Change password screen, enter and confirm your New password and select Accept.

 Lock and unlock

Employee can lock as needed. If you are stepping away from , we recommend you lock your screen. automatically locks when it is idle for the minutes set as Timeout minutes. A manager does not have to unlock the screen, you can unlock it with your password.

  • To lock menu, select Help > Screen lock. Enter your Password on the Screen lock prompt, and then select OK.

  • To unlock , simply enter your password on the Screen lock screen and select OK.

The last screen you accessed in is displayed.

Log out

Log out ends your session and stops tracking your employee number, as well as your time for hourly employees.

Log out of when you have completed working for the day.

  1. To log out from the menu or Invoice screen, select Logout > Logout/in.

  2. Select Logout, and then select Yes on the confirmation message.

    If you need to close , select Exit TAMS instead of Logout, and then select Yes on the confirmation message. You can also click the x on the top right of the Login screen to close after logging out.
